Professional employee scheduling system for contact center management.

== Description ==

PowerPlanner is a comprehensive employee scheduling solution designed specifically for contact centers and customer service teams. It provides advanced planning capabilities with intelligent pattern recognition, bulk operations, and real-time team management.

**Key Features:**

* **Advanced Planning Grid** - Visual weekly planning interface
* **Bulk Operations** - Mass edit multiple shifts at once
* **Pattern Recognition** - Automatic pattern detection and application
* **Team Management** - Real-time team status and availability
* **Smart Templates** - Configurable shift templates
* **Business Rules** - Automated rule enforcement
* **Export Functions** - CSV and Excel export capabilities
* **Mobile Responsive** - Works on all devices
* **News Ticker** - Team communication system
* **Dagstart Management** - Daily meeting chair assignments

**Perfect for:**
* Contact centers
* Customer service teams
* Call centers
* Help desk operations
* Any team-based scheduling needs

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/powerplanner/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the PowerPlanner menu item to configure and manage schedules
4. Add shortcodes to pages for employee and team views

== Frequently Asked Questions ==

= How do I set up employee views? =

Use the shortcode `[powerplanner_employee_view]` on any page to show individual employee schedules.

= How do I display team schedules? =

Use the shortcode `[powerplanner_team_view]` on any page to show the complete team schedule.

= Can I export schedules? =

Yes, PowerPlanner supports CSV and Excel export formats from the admin interface.

= Does it work on mobile devices? =

Yes, PowerPlanner is fully responsive and optimized for mobile use.
